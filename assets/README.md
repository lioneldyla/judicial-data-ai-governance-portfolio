## Overview

This directory contains visual assets, branding materials, logos, icons, and design resources associated with the **Data Governance for Intelligent Justice Systems** research programme.

Assets support the visual identity of the portfolio across publications, presentations, GitHub Pages, social media, and institutional communications.
