## FAIR Data Principles — Implementation

### Findable
- Each dataset assigned a persistent DOI via Zenodo
- Metadata registered with DataCite
- Dataset landing pages linked from this README

### Accessible
- Datasets hosted on Zenodo with open access where legally permissible
- Data dictionaries provided for all variables
- Access conditions documented where restricted data applies

### Interoperable
- Standard file formats: CSV, JSON, RDF where applicable
- Variable names follow judicial statistics nomenclature (CEPEJ standards where applicable)
- Codebooks provided in English and French

### Reusable
- CC BY 4.0 license for all open datasets
- Provenance metadata included in each dataset's `README.md`
- Methodology notes provided for derived indicators

## Data Management

All datasets are managed in accordance with the [Open Science Policy](../../OPEN_SCIENCE_POLICY.md) and the [Research Ethics](../../RESEARCH_ETHICS.md) guidelines. Personal or sensitive judicial data is never published; only aggregated, anonymised statistics are released.
