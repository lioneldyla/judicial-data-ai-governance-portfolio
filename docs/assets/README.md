# Assets

> **Category:** Documentation | **Version:** 1.0.0  
> **Last Updated:** July 2026

---

## Purpose

This directory stores static assets used across the repository documentation: logos, diagrams, figures, and images referenced in Markdown documents and the GitHub Pages site.

---

## Organisation

```
assets/
├── logos/          # Programme and institutional logos
├── diagrams/       # Framework and architecture diagrams
├── figures/        # Charts and figures from publications
└── icons/          # Badges and metadata icons
```

---

## Asset Standards

- **Format:** SVG preferred for diagrams and logos; PNG for raster figures (300 DPI minimum for print-quality)
- **Naming:** `[category]_[description]_[version].[ext]` (e.g., `diagram_jdgf_architecture_v1.svg`)
- **Attribution:** All third-party assets include source and license in an `ATTRIBUTION.md` file in the same subdirectory
- **Size:** Images optimised for web; source files retained separately if needed
