# Archive

> **Category:** Administration | **Version:** 1.0.0  
> **Last Updated:** July 2026

---

## Purpose

This directory archives superseded documents, draft materials no longer under active development, and deprecated versions of repository files. Archiving preserves intellectual history without cluttering the active repository.

---

## Archive Policy

A document is moved to the archive when:

1. It has been superseded by a newer version placed in its canonical location
2. It represented exploratory work that did not lead to a formal output
3. It was a placeholder or draft subsequently replaced by a complete document
4. A structural reorganisation displaced it from its original location

## Retrieval

Archived documents remain fully accessible. Their Git history is preserved. If a document in the archive needs to be reinstated as an active document, move it to the appropriate `docs/` subdirectory and update the CHANGELOG.
